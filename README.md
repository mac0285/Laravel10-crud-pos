# Sitem Penjualan dengan Laravel 9
 
## Installation
1. Clone Repo
    ```console
    git clone  https://github.com/mac0285/Laravel10-crud-pos.git
    ```

2. Run
   ```console
   php artisan key:generate
   ```
   ```console
   php artisan migrate
   or
   php artisan migrate:refresh
   ```
   ```console
   php artisan db:seed
   ```

3. Run server
   ```console
   php artisan serve
   ```
